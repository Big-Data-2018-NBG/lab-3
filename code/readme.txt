To execute the algorithms filenames will need to be changed within the code. 
To change the file names open the code and change the desired files that will be used as input.

To run:
Ensure MrJob is installed for python.

Type the following command into terminal:
python <code file name> <input 1> <input 2>

an example to run algorithm 1 for the first set of matrices is:
python algorithm1-two-step.py outA1.list outB1.list
